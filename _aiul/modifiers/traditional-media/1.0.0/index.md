# Traditional Media v1.0.0

## Overview

The **Traditional Media** domain covers analog and hand-made creative workâ€”such as drawing, painting, sculpture, printmaking, and other tactile artistic practices.

- Painting (acrylic, oil, watercolor, etc.)
- Sculpture and 3D physical forms
